# 行事曆同步設定完整指南

將 Orlo 與你的行事曆整合，讓時間管理更加順暢。本指南涵蓋所有主流行事曆的設定方法。

## 支援的行事曆平台

### Google Calendar
- **最佳整合體驗**
- 即時雙向同步
- 多日曆支援
- 自動衝突檢測

### Microsoft Outlook / Office 365
- 企業環境首選
- Exchange 伺服器支援
- 團隊共享行事曆
- 會議室預訂整合

### Apple Calendar (iCloud)
- macOS/iOS 原生支援
- 家庭共享整合
- Siri 語音控制
- 跨裝置同步

### 其他平台
- Yahoo Calendar
- Thunderbird
- CalDAV 標準協議

## ⚙Google Calendar 設定步驟

### 第一次連接

1. **進入設定頁面**
   - 點擊右上角個人頭像
   - 選擇「設定」→「行事曆整合」

2. **授權連接**
   - 點擊「連接 Google Calendar」
   - 登入你的 Google 帳戶
   - 授權 Orlo 存取權限

3. **選擇同步範圍**
   - 主要行事曆：建議開啟
   - 工作行事曆：依需求選擇
   - 私人行事曆：可選擇性同步

4. **設定同步規則**
   - 同步時間範圍：過去 1 週，未來 4 週
   - 更新頻率：每 15 分鐘
   - 衝突處理：Orlo 優先 / Calendar 優先

### 進階設定選項

**自動 TimeBox 建立**
```
- 會議自動建立對應 TimeBox
- 為會議前後預留緩衝時間
- 排除特定類型的事件（如生日提醒）
```

**隱私控制**
```
只同步空閒/忙碌狀態
同步事件標題但隱藏詳細內容
完全同步所有資訊
```

## 🏢 Outlook/Office 365 設定

### 企業環境設定

1. **管理員授權**（如果需要）
   - 聯繫 IT 部門
   - 申請第三方應用權限
   - 獲取必要的安全許可

2. **個人帳戶連接**
   - 設定 > 行事曆整合 > Outlook
   - 輸入公司郵箱地址
   - 完成 OAuth2 驗證

3. **Exchange 設定**
   - 伺服器位址設定
   - SSL 加密確認
   - 連線測試驗證

### 特殊功能

**會議室預訂**
- 自動檢查會議室可用性
- 整合到 TimeBox 安排
- 衝突提前提醒

**團隊行事曆**
- 查看團隊成員空檔
- 共享專案時程
- 協作會議安排

## Apple Calendar 整合

### iCloud 帳戶設定

1. **啟用 iCloud 同步**
   - iOS：設定 > [你的名字] > iCloud > 行事曆
   - macOS：系統偏好設定 > Apple ID > iCloud

2. **Orlo 連接設定**
   - 使用 Apple ID 登入
   - 授權行事曆存取
   - 選擇要同步的行事曆

### iOS 特有功能

**Siri 語音整合**
```
"Hey Siri，今天的 Orlo 安排是什麼？"
"Hey Siri，明天上午有空嗎？"
```

**Widget 支援**
- 今日行程一覽
- 下個 TimeBox 提醒
- 快速新增任務

## 雙向同步機制

### 從 Calendar 到 Orlo
- 新增的行程自動建立 TimeBox
- 時間變更自動更新
- 取消的事件自動移除

### 從 Orlo 到 Calendar
- TimeBox 建立對應行程
- 時間調整即時同步
- 標籤轉換為行程分類

### 衝突處理規則

**時間衝突**
```
規則 1：行事曆約會優先
規則 2：Orlo TimeBox 自動調整
規則 3：提醒用戶手動確認
```

**重複事件**
```
- 智能去重機制
- 合併相似事件
- 保留重要信息
```

## 進階自訂設定

### 同步篩選規則

**按關鍵字篩選**
- 包含特定詞彙的事件
- 排除垃圾郵件約會
- 工作相關事件優先

**按參與者篩選**
- 只同步特定聯絡人的邀請
- 排除大型群組會議
- VIP 聯絡人優先處理

### 自動化規則

**會議準備時間**
```
IF: 會議時長 > 1 小時
THEN: 前 30 分鐘自動安排準備時間
```

**通勤時間計算**
```
IF: 會議地點 ≠ 辦公室
THEN: 自動加入通勤時間 TimeBox
```
