# 快速上手指南

歡迎使用 Orlo！這份指南將在 5 分鐘內帶你了解 Orlo 的核心功能。

## 什麼是 Orlo？

Orlo 是你的 AI 時間管理助手，就像高階主管的個人秘書，幫你：

- **規劃時間**：將待辦事項轉化為具體的時間安排
- **減少決策**：AI 自動建議最佳的時間配置
- **平衡生活**：智能管理工作與生活的界限

## 第一步：註冊並登入

1. 前往 [my.orlo.cc](https://my.orlo.cc)
2. 使用 Google 帳號快速註冊
3. 完成基本資料設定

</div>

## 第二步：建立你的第一個 TimeBox

TimeBox 是 Orlo 的核心概念，將抽象的「待辦事項」轉化為具體的「何時做」。

### 如何建立 TimeBox：

1. **點擊「新增 TimeBox」**
2. **輸入任務內容**：例如「準備週會簡報」
3. **設定時間長度**：AI 會建議合適的時間，你也可以調整
4. **選擇時間段**：讓 AI 找出最佳時間，或手動指定

### TimeBox vs 一般待辦清單：

| 一般待辦清單 | Orlo TimeBox |
|-------------|--------------|
| "準備簡報" | "週三 10:00-12:00 準備簡報" |
| 模糊、容易拖延 | 具體、容易執行 |
| 無時間感 | 有明確時間界限 |

## 第三步：與 AI 助手對話

Orlo 的 AI 助手可以幫你：

- **優化時間安排**：「我明天的行程太滿了」
- **處理突發狀況**：「臨時有個會議，怎麼調整？」
- **生活建議**：「如何在忙碌的工作中保持健康？」

### 實用對話範例：

```
你：明天有三個重要任務，但感覺時間不夠
AI：我來幫你分析。根據你的行程，建議將「資料分析」移到週四上午，這樣明天可以專注在簡報和會議上。你覺得如何？
```

## 第四步：同步你的生活

### 連接外部行事曆

- **Google Calendar**：自動同步會議和約會
- **Apple Calendar**：iPhone 用戶的最佳選擇
- **Outlook**：商務用戶的整合方案

### 設定工作偏好

- **最佳工作時段**：告訴 Orlo 你何時效率最高
- **休息設定**：確保適當的休息時間
- **專注模式**：設定免打擾時段

## 恭喜！你已經準備就緒

1. 註冊並登入
2. 建立第一個 TimeBox
3. 與 AI 助手互動
4. 連接外部服務
