# AI 助手完整功能指南

Orlo 的 AI 助手是你的個人時間管理專家，能夠學習你的工作模式，提供智能建議，並自動優化你的時間安排。

## 🤖 AI 助手能做什麼

### 智能任務分解
- 將大型專案拆解為可執行的小任務
- 估算每個子任務的時間需求
- 建議最佳的執行順序

### 時間預估優化
- 根據歷史數據預測任務時間
- 學習你的工作節奏和效率模式
- 動態調整時間估算

### 優先級智能建議
- 分析任務的重要性和緊急性
- 考慮截止日期和依賴關係
- 提供個性化的優先級排序

### 工作模式學習
- 識別你的高效時段
- 了解不同任務類型的能量需求
- 建立個人化的工作模式

## 💬 有效對話技巧

### 基本對話原則
- 使用自然語言，無需特殊指令
- 提供具體的上下文信息
- 明確表達你的需求和限制

### 常用指令範例

**任務規劃**
```
"我明天有個重要簡報，需要 3 小時準備時間，什麼時候安排最好？"
"幫我把'完成季度報告'這個任務分解成小步驟"
```

**時間優化**
```
"我的行程太滿了，有什麼可以調整的嗎？"
"週三下午有什麼時間可以安排深度工作？"
```

**習慣建立**
```
"如何在忙碌的工作中保持規律運動？"
"幫我建立晨間例行公事的時間安排"
```

## ⚙️ 個性化設定

### AI 學習偏好
- 工作風格偏好（專注 vs 多工）
- 時間估算傾向（保守 vs 積極）
- 中斷容忍度設定

### 自動化程度
- 完全自動：AI 直接執行建議
- 半自動：AI 提供選項讓你選擇
- 輔助模式：僅提供建議不自動執行

### 溝通風格
- 正式商務風格
- 友善對話風格
- 簡潔高效風格

## 🎯 進階應用技巧

### 情境化請求
針對不同情況使用特定的請求方式：

**緊急狀況處理**
```
"突然有個緊急會議，請幫我重新安排今天的行程"
```

**長期規劃**
```
"接下來兩週我要準備產品發表會，請幫我制定完整計劃"
```

**能量管理**
```
"我今天感覺很累，有什麼輕鬆的任務可以做？"
```

### 學習反饋機制
- 完成任務後提供反饋
- 標記預估時間的準確性
- 評估建議的實用性

### 協作增強
- 與團隊共享 AI 建議
- 集體決策輔助
- 會議時間協調
