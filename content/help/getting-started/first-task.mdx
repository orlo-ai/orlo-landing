# 建立你的第一個任務

在 Orlo 中，每個任務都會轉化為具體的 TimeBox。讓我們一步一步學習如何建立高效的任務安排。

## 📝 任務建立流程

### 1. 點擊「新增任務」

在主畫面找到藍色的「+」按鈕，或是直接點擊「新增 TimeBox」。

### 2. 填寫任務基本資訊

**任務標題**
- 用動詞開頭：「完成」、「準備」、「檢視」
- 具體明確：避免模糊的描述

✅ **好的範例**
- "完成 Q3 銷售報告"
- "準備明天的客戶簡報"
- "整理本週會議記錄"

❌ **避免的範例**
- "處理一些工作"
- "開會相關的事"
- "雜事"

### 3. 設定預估時間

Orlo 會根據任務類型建議時間：

| 任務類型 | 建議時間 |
|----------|----------|
| 電子郵件處理 | 30-60 分鐘 |
| 報告撰寫 | 2-4 小時 |
| 會議準備 | 30-90 分鐘 |
| 資料分析 | 1-3 小時 |

<div className="bg-green-50 border-l-4 border-green-500 p-4 my-4">
💡 **專業提示**：剛開始使用時，寧可高估時間。Orlo 會學習你的工作節奏，逐漸提供更準確的建議。
</div>

### 4. 選擇優先級

- **高優先級**：今天必須完成
- **中優先級**：本週內完成
- **低優先級**：有空時處理

### 5. 添加標籤和分類

使用標籤幫助 Orlo 更好地理解你的工作模式：

- **工作類型**：#會議 #寫作 #分析 #溝通
- **專案分類**：#產品開發 #行銷活動 #客戶服務
- **能量需求**：#高專注 #例行工作 #創意思考

## 🤖 讓 AI 協助最佳化

建立任務後，你可以問 AI：

### 時間安排建議
```
「這個任務什麼時候做最合適？」
```

### 拆解複雜任務
```
「這個任務太大了，可以幫我分解嗎？」
```

### 提醒相關工作
```
「做這個任務前，我需要準備什麼？」
```

## ⚙️ 進階設定選項

### 重複任務
- 每日：處理 email、晨間計劃
- 每週：週會準備、週報撰寫
- 每月：月度檢討、報告彙整

### 緩衝時間
- 會議前：5-10 分鐘準備時間
- 任務間：適當的轉換時間
- 意外處理：預留彈性時間

### 環境需求
- 需要安靜環境
- 需要特定工具或資料
- 適合的工作地點

## 📊 任務完成後的檢討

完成任務時，Orlo 會詢問：

1. **實際用時 vs 預估時間**
2. **任務難度感受**
3. **完成品質評估**
4. **遇到的障礙**

這些回饋幫助 AI 更準確地安排未來任務。

## 🔄 任務狀態管理

### 任務狀態說明

- **計劃中**：尚未開始
- **進行中**：正在執行
- **暫停**：臨時中斷
- **已完成**：成功完成
- **已取消**：不再需要

### 狀態變更操作

- **開始任務**：點擊「開始」進入專注模式
- **暫停任務**：處理突發狀況時使用
- **完成任務**：填寫完成回饋

## 🎯 常見問題

### Q: 任務太多，不知道先做哪個？
A: 讓 AI 協助排序。Orlo 會考慮截止日期、重要性和你的能量水平。

### Q: 任務中途被打斷怎麼辦？
A: 使用「暫停」功能，Orlo 會記錄進度，並建議最佳的重啟時間。

### Q: 預估時間總是不準怎麼辦？
A: 持續記錄實際時間，Orlo 會學習你的工作模式，預估會越來越準確。
